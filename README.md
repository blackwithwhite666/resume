Resume
======

Direct link
-----------

[Download!](https://github.com/blackwithwhite666/resume/raw/master/black-resume.pdf)

How to compile
--------------

To compile it you should run:

> lualatex black-resume.tex

Installation
------------

For deb-based system ensure that those packages installed:

* ttf-sil-charis
* ttf-liberation
* ttf-inconsolata
* texlive-full
* texlive-luatex
